# TeamFinder

Платформа для поиска команды и участников в IT-проекты. Разработчики могут находить проекты по своим навыкам, а владельцы проектов — приглашать подходящих специалистов.

## Возможности

- Регистрация и авторизация пользователей
- Создание и редактирование профиля (аватар, навыки, контакты)
- Публикация проектов с указанием требуемых навыков
- Поиск проектов по навыкам
- Добавление проектов в избранное
- Участие в проектах
- Просмотр участников и авторов проектов
- Фильтрация пользователей по участию в проектах

## Технологии

- **Backend**: Django 5.2, Python 3.10
- **Database**: PostgreSQL
- **Frontend**: HTML, CSS (вариант шаблонов v3)
- **Containerization**: Docker, Docker Compose
- **Libraries**: Pillow (генерация аватаров), python-decouple (настройки)

## Запуск проекта

### 1. Клонируйте репозиторий

```bash
git clone <url-репозитория>
cd team-finder-ad-main
```

### 2. Запустите виртуальное окружение

```bash
sourse venv/Scripts/activate
```

### 3. Запустите Docker (базу данных):

```bash
docker compose up -d
```

### 4. Примените миграции

```bash
python manage.py migrate
```

### 5. Создайте суперпользователя

```bash
python manage.py createsuperuser
```

### 6. Запустите сервер
```bash
python manage.py runserver
```

### 7. Откройте в браузере: http://localhost:8000
